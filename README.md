# React To-Do App

A simple and responsive To-Do List application built using React.js and Tailwind CSS.

## Features

- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Data persistence using Local Storage
- Responsive UI
- UUID-based unique task IDs

## Tech Stack

- React.js
- Tailwind CSS
- JavaScript
- UUID
- React icons
- Local Storage API
